    Extract information from the post filename.

    name - The String filename of the post file.

    Returns nothing.

    # File lib/jekyll/draft.rb, line 26
def process(name)
  m, slug, ext = *name.match(MATCHER)
  self.date = File.mtime(File.join(@base, name))
  self.slug = slug
  self.ext = ext
end
